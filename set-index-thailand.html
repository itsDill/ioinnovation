About the SET Index

The Stock Exchange of Thailand SET Index is the main benchmark index for the Thai stock market, comprising all stocks listed on the main board. Thailand's economy is driven by tourism, agriculture, automotive manufacturing, and financial services, making the SET Index a comprehensive gauge of the country's economic performance.
